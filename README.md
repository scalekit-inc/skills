# Skills

Reusable skills for building with Scalekit across coding agents.

This repo is a collection of agent-friendly instructions that help with repeatable Scalekit workflows. Each skill is designed to give an agent a focused job, a clear operating boundary, and a predictable way to help.

## What This Repo Contains

- reusable skills for common Scalekit tasks
- task-specific instructions in `SKILL.md` files
- supporting agent actions and templates as the repo grows

## Repo Structure

- `agent-actions/`: agent-auth and actions skills

Each skill lives in its own folder and is usually centered around a `SKILL.md` file.

## How To Use

Pick the skill folder you need, then install or copy that skill into the agent you use, such as Claude Code, Codex, or GitHub Copilot.

After that, ask the agent to use the skill by name and provide the inputs required for your task.

## Goal

Keep Scalekit workflows simple, reusable, and easier for agents to execute correctly.
